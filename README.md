# Dynamic Post Composer

## 📌 Aim

To design and develop a dynamic post composer interface supporting multiple platforms with constraint validation using React and TypeScript.

---

## 🎯 Objectives

- Understand multi-platform content handling.
- Implement real-time validation mechanisms.
- Design a responsive and user-friendly UI.
- Build reusable React components.

---

## 🚀 Features

- Multi-platform selection
- Character count for each platform
- Platform-specific validation
- Responsive user interface
- Built using React + TypeScript
- Component-based architecture

---

## 🛠 Technologies Used

- React
- TypeScript
- Vite
- HTML5
- CSS3
